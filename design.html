Video Games

Personal Project - First Person Shooter UI


Objective:

Design a UI prototype of a first person shooter game with branching story paths, special abilities, and a open-world map to explore.

I personally wanted to explore how a video game user interface was designed with interactive elements.

1) Health and Stamina Bars (Green = Health; Orange = Stamina)

2) Objectives - Primary and Optional (Players can retreive a secret weapon by discovering the optional weapon cache)

3) Special abilities activated with button prompts (More abilities can be discovered by exploring)

4) Wireframe Assault Rifle Model

5) On-Screen Destination Marker for the Primary Objective

6) Radar Map

7) Weapon Crosshair

8) Weapon Name and Ammo Count

Designed in Adobe Illustrator
